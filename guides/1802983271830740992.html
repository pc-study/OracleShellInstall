统信 UOS V20 1050(d) 一键安装 Oracle 11GR2 单机

原文来源：墨天轮 作者：Lucifer

前言

Oracle 一键安装脚本，演示 统信 UOS V20 1050(d) 一键安装 Oracle 11GR2 单机（全程无需人工干预）。

⭐️ 脚本下载地址：Shell脚本安装Oracle数据库

前置准备

  • 1、安装好操作系统，建议安装图形化
  • 2、配置好网络
  • 3、挂载本地 ISO 镜像源
  • 4、上传软件安装包（安装基础包，补丁包）
  • 5、上传一键安装脚本：OracleShellInstall

安装命令

使用标准生产环境安装参数：

# 根据脚本 README 或者 -h 命令提示，编辑好一键安装命令，进入 /soft 目录执行安装：
./OracleShellInstall -lf ens33 `# local ip ifname`\
-n uos1050d `# hostname`\
-op oracle `# oracle password`\
-d /u01 `# software base dir`\
-ord /oradata `# data dir`\
-o lucifer `# dbname`\
-dp 'Passw0rd#PST' `# sys/system password`\
-ds AL32UTF8 `# database character`\
-ns AL16UTF16 `# national character`\
-redo 100 `# redo size`\
-opd Y `# optimize db`
