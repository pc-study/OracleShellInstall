еүҚиЁҖ

Oracle дёҖй”®е®үиЈ…и„ҡжң¬пјҢжј”зӨә йә’йәҹ Kylin V10 SP3 дёҖй”®е®үиЈ… Oracle 12CR2 RAC иҝҮзЁӢпјҲе…ЁзЁӢж— йңҖдәәе·Ҙе№Ійў„пјүгҖӮ

е®үиЈ…еҮҶеӨҮ

  • 1гҖҒзі»з»ҹз»„жүҖжңүиҠӮзӮ№еқҮе®үиЈ…еҘҪж“ҚдҪңзі»з»ҹпјҲж”ҜжҢҒжңҖе°ҸеҢ–е®үиЈ…пјү
  • 2гҖҒзҪ‘з»ңз»„жүҖжңүиҠӮзӮ№еқҮй…ҚзҪ®еҘҪдё»жңәзҪ‘з»ңпјҢиҮіе°‘йңҖиҰҒдёҖз»„е…¬зҪ‘ IP ең°еқҖе’ҢдёҖз»„еҝғи·і IP ең°еқҖ
  • 3гҖҒеӯҳеӮЁз»„жүҖжңүиҠӮзӮ№еқҮй…ҚзҪ®е№¶еңЁдё»жңәеұӮжҢӮиҪҪеҘҪ ASM зЈҒзӣҳпјҢиҮіе°‘йңҖиҰҒдёҖз»„ OCR е’Ң DATA зЈҒзӣҳз»„пјҢиҷҡжӢҹеҢ–зҺҜеўғйңҖиҰҒзЎ®дҝқе·ІејҖеҗҜзЈҒзӣҳзҡ„ UUID
  • 4гҖҒDBA еҸӘйңҖиҰҒеңЁдё»иҠӮзӮ№еҲӣе»әиҪҜд»¶зӣ®еҪ•пјҡmkdir /soft
  • 5гҖҒDBA еҸӘйңҖиҰҒеңЁдё»иҠӮзӮ№дёҠдј  Oracle е®үиЈ…д»ӢиҙЁпјҲеҹәзЎҖеҢ…пјҢиЎҘдёҒеҢ…пјүеҲ° /soft зӣ®еҪ•дёӢпјҢе…¶д»–иҠӮзӮ№ж— йңҖд»»дҪ•ж“ҚдҪң
  • 6гҖҒDBA еҸӘйңҖиҰҒеңЁдё»иҠӮзӮ№дёҠдј  Oracle дёҖй”®е®үиЈ…и„ҡжң¬еҲ° /soft зӣ®еҪ•дёӢпјҢжҺҲдәҲи„ҡжң¬жү§иЎҢжқғйҷҗпјҡchmod +x OracleshellInstall
  • 7гҖҒDBA жүҖжңүиҠӮзӮ№еқҮжҢӮиҪҪдё»жңә ISO й•ңеғҸпјҢиҝҷйҮҢеҸӘйңҖиҰҒ mount дёҠеҚіеҸҜпјҲиҝҷдёӘеҫҲз®ҖеҚ•пјҢдёҚдәҶи§Јзҡ„еҸҜд»ҘзҷҫеәҰдёӢпјү
  • 8гҖҒж №жҚ®и„ҡжң¬е®үиЈ…и„ҡжң¬д»ҘеҸҠе®һйҷ…жғ…еҶөпјҢй…ҚзҪ®еҘҪи„ҡжң¬зҡ„е®үиЈ…еҸӮж•°пјҢеңЁдё»иҠӮзӮ№зҡ„ /soft зӣ®еҪ•дёӢжү§иЎҢдёҖй”®е®үиЈ…еҚіеҸҜгҖӮ
