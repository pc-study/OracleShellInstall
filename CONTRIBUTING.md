# Contributing to OracleShellInstall

感谢你对 OracleShellInstall 的关注！

## 如何贡献

### 报告 Bug
- 在 [Issues](https://github.com/pc-study/OracleShellInstall/issues) 中提交问题
- 请包含：操作系统版本、Oracle 版本、执行的命令、错误日志

### 功能建议
- 在 Issues 中提交功能请求，标注 `[Feature Request]`

### 提交代码
1. Fork 本仓库
2. 创建你的功能分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送到分支：`git push origin feature/your-feature`
5. 提交 Pull Request
